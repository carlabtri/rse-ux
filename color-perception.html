Color Perception Tool

A web-based test that uses ML sampling methods to iteratively measure a user's sensitivity to color — delivering a probability plot that shows how accurately they perceive minor variations of the same hue.

▶ Demo✦ Figma⬡ Jira Story

Overview

An internal web app that uses ML sampling methods — including Bayesian sampling — to iteratively identify a user's sensitivity to a particular color. At the end of the test, users receive a probability plot showing how accurately they perceive minor variations of the same hue.

Goal

Enable TRI researchers to test individual hue sensitivity so they can understand how accurately participants perceive slight differences between colors.

Phase I Scope

  • Color selection input fields
  • Preview and testing range
  • Instructions and color gradient block
  • Practice grid
  • Page loading states

Welcome Page

An introductory screen that presents instructions to orient users before the test begins.

Gradient Test

Users are shown a color gradient and asked to identify the outlier — testing their ability to distinguish subtle hue variations within a controlled range.

Practice Screens

Four color squares labeled 1–4 are displayed side by side. Users identify the outlier using their keyboard, building familiarity with the test format before the scored session begins.

Results & Probability Plot

At the end of the session, a probability plot is generated from the Bayesian sampling data — visualizing the user's color sensitivity threshold and how reliably they identified minor variations.
