LLM2CAD

A tool that transforms text, image, and program prompts into CAD files — bridging human creativity and precision engineering through AI-powered generation.

▶ Demo✦ Figma⬡ Jira Epic

Overview

LLM2CAD transforms text, image, and program prompts into CAD files. Users describe what they want — in natural language, as an image reference, or as a program — and the tool produces precise CAD output.

Design Focus

  • Intuitive natural language input form
  • Clear CAD viewer with exploration controls
  • Feedback loop between prompt and output
