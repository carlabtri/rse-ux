Ops Chatbot

An internal AI-powered tool allowing users to select between three underlying LLMs for day-to-day tasks, spanning an evaluation dashboard and a voice-enabled mobile interface.

▶ Demo✦ Figma⬡ Jira Epic

Part 1 — Chatbot Interface

AI-powered chatbot integrated into TRI's intranet.

Part 2 — Evaluation Tool

Full UI for an internal LLM evaluation platform enabling TRI researchers to evaluate, compare, and analyze language model outputs through a structured dashboard and thread-detail view.

Key Feature

Users can switch between three underlying LLMs — the UX needed to make this selection clear, contextual, and non-disruptive to the workflow.
