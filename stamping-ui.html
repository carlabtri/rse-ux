Stamping UI — Robotics

UI for a defect detection system using computer vision and ML to monitor defects in auto panel stamping — a browser-based app for filtering and viewing data from 15 cameras.

▶ Demo

Overview

A UI for a defect detection system that leverages computer vision and ML to monitor and manage defects in the auto panel stamping process.

Context

A browser-based application allowing users to filter and view a large collection of data generated by 15 cameras across the stamping line.

Challenge

Presenting high-volume, multi-camera data in a way that lets operators quickly identify and act on defects without cognitive overload.
