Overview

A graphical user interface for E&M's AMDD team to create, save, load, and analyze graphical models — using a node-based graph editor for building and visualizing mechanism models.

Challenge

The tool is used by engineers, not designers. The graph interface needed to be powerful and precise while remaining discoverable and efficient for non-UX users.

Approach

  • Multiple iterative design sprints
  • Focused on discoverability and interaction clarity
  • Close collaboration with engineering on feasibility
